# AI Chat Application

An intelligent chat application that leverages OpenAI's GPT-4 to provide natural, context-aware conversations.

## Features

- **Real-Time Messaging**: WebSocket-based real-time communication
- **AI Personas**: Customizable AI personalities for different use cases
- **Conversation History**: Persistent chat history with search functionality
- **Markdown Support**: Rich text formatting in messages
- **Code Highlighting**: Syntax highlighting for code snippets
- **Voice Input**: Speech-to-text integration

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **AI**: OpenAI GPT-4 API
- **Real-Time**: Socket.io
- **Deployment**: Docker, AWS

## Challenges Overcome

Implementing efficient token management for API calls, handling WebSocket connections at scale, and creating a responsive, accessible chat interface were key challenges in this project.
