**Repo Hunter** is a frontend-only project that integrates with the **GitHub GraphQL API** using **Apollo Client**.  
It allows users to enter a GitHub username and view detailed repository insights — including star counts, commit totals, creation dates, and even the top 5 stargazers.  

The interface features elegant transitions and subtle visual effects powered by **Aceternity UI**, combining aesthetics and interactivity for an engaging developer experience.  

---

## Core Features  


- **Search GitHub Users** and list their public repositories
- **View stars, stargazers, and commits** for each repo  
- **Formatted dates** — includes “last updated X days ago”  
- **Fully responsive modal and layout** for mobile and desktop  
- **Typewriter animations** and **sparkle effects** for dynamic feedback  

---

## Tech Stack  
- **React + TypeScript + Vite** — frontend foundation  
- **Apollo Client + GitHub GraphQL API** — efficient data fetching  
- **Tailwind CSS** — responsive and utility-first styling  
- **Aceternity UI** — beautiful, pre-built animations (TypewriterEffect, SparklesCore, TracingBeam)  

---

## Key Takeaways  
- Learned to integrate **GraphQL APIs** efficiently using Apollo Client’s caching system.  
- Understood the difference between **query efficiency** and REST overfetching.  
- Practiced **responsive UI design** with Tailwind breakpoints.
