# Task Management Dashboard

A powerful task management tool designed for teams to collaborate effectively.

## Features

- **Drag & Drop**: Intuitive drag-and-drop interface for task organization
- **Team Collaboration**: Invite team members and assign tasks
- **Real-Time Updates**: See changes as they happen
- **Custom Boards**: Create unlimited boards with custom workflows
- **Due Dates & Reminders**: Never miss a deadline
- **File Attachments**: Attach files and images to tasks
- **Activity Timeline**: Track all changes and updates

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Drag & Drop**: DnD Kit
- **Backend**: Supabase (PostgreSQL + Real-time)
- **Authentication**: Supabase Auth
- **File Storage**: Supabase Storage
- **Deployment**: Vercel

## Design Philosophy

The project focuses on simplicity and usability, with keyboard shortcuts, accessibility features, and a clean, distraction-free interface.
